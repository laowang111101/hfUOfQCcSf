# 学生宿舍信息管理系统 java914

## 项目概述

学生宿舍信息管理系统是基于前后端分离架构，为学校宿舍管理部门提供全面的信息管理解决方案。系统支持系统管理员、宿舍管理员与学生三种角色，实现宿舍管理的高效与智能化。

## 技术栈

- **开发环境：**
  - IntelliJ IDEA
  - JDK 1.8
  - Maven
  - Node.js
- **技术框架：**
  - Java
  - SpringBoot
  - MyBatis-Plus
  - Vue

## 数据库

- MySQL

## 功能模块

### 系统管理员模块

- 学生信息管理
- 宿管信息管理
- 楼宇信息管理
- 房间信息管理
- 公告信息管理
- 报修信息管理
- 调宿信息管理
- 访客管理
- 个人信息

### 宿舍管理员模块

- 学生信息管理
- 楼宇信息管理
- 房间信息管理
- 公告信息管理
- 报修信息管理
- 调宿信息管理
- 访客管理
- 个人信息

### 学生模块

- 学生首页
- 查看公告
- 我的宿舍
- 申请调宿
- 报修申请
- 个人信息

## 运行环境

- 服务器：推荐使用 Linux 服务器，Windows 服务器亦可
- 数据库：MySQL 5.7 或更高版本
- 前端：支持现代浏览器，建议使用 Chrome 或 Firefox
- 后端：JDK 1.8 或更高版本

## 核心亮点

- **灵活的角色权限管理：** 系统根据不同角色分配不同权限，保证系统安全性和管理灵活性。
- **高效的数据管理：** 使用 MyBatis-Plus，简化了数据访问层的开发，提高了开发效率。
- **模块化设计：** 系统功能模块清晰，易于扩展和维护。
